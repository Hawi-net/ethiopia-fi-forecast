# Ethiopia Financial Inclusion Forecast

## Project Overview

The Ethiopia Financial Inclusion Forecast project aims to analyze historical financial inclusion data in Ethiopia and identify factors that influence access and usage of financial services.

The project explores trends in:
- Account ownership
- Digital financial services
- Mobile money adoption
- Infrastructure development
- Policy and market events

The goal is to prepare data-driven insights and forecasting models that can help understand future financial inclusion patterns in Ethiopia.

---

## Project Objectives

- Explore and understand Ethiopia financial inclusion datasets
- Enrich existing data with additional observations, events, and impact relationships
- Perform exploratory data analysis (EDA)
- Identify factors influencing financial access and usage
- Build a foundation for future forecasting and impact modeling
